# Public Assets

This folder contains static assets that will be served directly by Vite.

## Logo

Place your logo file here with one of these names:
- `logo.png` (recommended)
- `logo.svg` (alternative)

The logo will be displayed in the application header. If no logo is found, the Crown icon will be used as a fallback.

### Recommended Logo Specifications:
- **Format**: PNG or SVG
- **Size**: 48x48 pixels (or higher for retina displays)
- **Background**: Transparent (for best appearance on gradient header)
- **Aspect Ratio**: 1:1 (square)
